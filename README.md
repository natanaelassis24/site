# Bike Catalogue

Catálogo de bicicletas/motos em React inspirado em um layout editorial de catálogo.

## Tecnologias

- React
- Vite
- CSS puro

## Como rodar

```powershell
npm.cmd install
npm.cmd run dev
```

## Build de produção

```powershell
npm.cmd run build
```

## Deploy na Vercel

1. Envie o projeto para o GitHub.
2. Importe o repositório na Vercel.
3. Use o comando de build `npm run build`.
4. Use `dist` como pasta de saída.

## Observações

- As imagens dos cards ficam na pasta `public`.
- A interface tem filtros, busca e paginação simples.
